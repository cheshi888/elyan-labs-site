React Counter Demo

This page demonstrates a simple React counter component created for the Elyan Labs bounty task.

Increment Counter

Click the "+" button to increase the counter value.

Decrement Counter

Click the "-" button to decrease the counter value.

Reset Function

Click "Reset" to set the counter back to zero.

Visual Feedback

The counter color changes based on its value.

Component Code

The React component is implemented using functional components with hooks:

// React Counter Component const Counter = () => { const [count, setCount] = React.useState(0); const increment = () => setCount(count + 1); const decrement = () => setCount(count - 1); const reset = () => setCount(0); // Determine color based on count value const getCounterColor = () => { if (count > 0) return '#48bb78'; // Green for positive if (count < 0) return '#f56565'; // Red for negative return '#667eea'; // Blue for zero }; return React.createElement('div', { style: { textAlign: 'center', padding: '30px', borderRadius: '15px', backgroundColor: '#f7fafc' } }, React.createElement('h2', { style: { color: '#667eea', marginBottom: '20px' } }, 'React Counter Component'), React.createElement('div', { style: { fontSize: '5rem', fontWeight: 'bold', color: getCounterColor(), margin: '30px 0', transition: 'color 0.3s ease' } }, count), React.createElement('div', { style: { display: 'flex', justifyContent: 'center', gap: '15px', marginBottom: '20px' } }, React.createElement('button', { onClick: decrement, style: { padding: '12px 24px', fontSize: '1.2rem', backgroundColor: '#f56565', color: 'white', border: 'none', borderRadius: '8px', cursor: 'pointer', transition: 'transform 0.2s' }, onMouseOver: (e) => e.target.style.transform = 'scale(1.05)', onMouseOut: (e) => e.target.style.transform = 'scale(1)' }, '- Decrement'), React.createElement('button', { onClick: reset, style: { padding: '12px 24px', fontSize: '1.2rem', backgroundColor: '#a0aec0', color: 'white', border: 'none', borderRadius: '8px', cursor: 'pointer', transition: 'transform 0.2s' }, onMouseOver: (e) => e.target.style.transform = 'scale(1.05)', onMouseOut: (e) => e.target.style.transform = 'scale(1)' }, 'Reset'), React.createElement('button', { onClick: increment, style: { padding: '12px 24px', fontSize: '1.2rem', backgroundColor: '#48bb78', color: 'white', border: 'none', borderRadius: '8px', cursor: 'pointer', transition: 'transform 0.2s' }, onMouseOver: (e) => e.target.style.transform = 'scale(1.05)', onMouseOut: (e) => e.target.style.transform = 'scale(1)' }, '+ Increment') ), React.createElement('div', { style: { marginTop: '20px', color: '#4a5568', fontSize: '0.9rem' } }, React.createElement('p', null, `Current value: ${count}`), React.createElement('p', null, `Value is ${count > 0 ? 'positive' : count < 0 ? 'negative' : 'zero'}`), React.createElement('p', null, `Absolute value: ${Math.abs(count)}`) ) ); };

How It Works

This React component demonstrates:

  • React State Management: Using useState hook for counter state
  • Event Handling: Button click handlers for increment, decrement, and reset
  • Conditional Rendering: Color changes based on counter value
  • Component Composition: Building UI with React.createElement
  • Inline Styling: CSS-in-JS approach for component styling
  • Interactive Feedback: Hover effects and transitions
