# Elyan Labs Site

A collection of web pages and resources for Elyan Labs projects and experiments.

## Overview

This repository contains various web pages, demos, and utilities created for Elyan Labs. It serves as a playground for web development experiments, React components, and bounty task demonstrations.

- **React Components**: Interactive React components with modern hooks
- **HTML/CSS Demos**: Clean, responsive web pages

## Getting Started

### Quick Start

Simply clone the repository and open any HTML file in your browser:

```bash
git clone https://github.com/Scottcjn/elyan-labs-site.git
cd elyan-labs-site
open index.html  # Or double-click the file in your file explorer
```

## React Components

### Counter Component

The `react-counter.html` file contains a fully functional React counter component with:

- **State Management**: Uses React useState hook
- **Interactive Controls**: Increment, decrement, and reset buttons
- **Visual Feedback**: Color changes based on counter value
- **Responsive Design**: Works on all device sizes

To use it:
1. Open `react-counter.html` in any modern browser
2. The page loads React and Babel from CDN
3. No build process required

## Configuration

### config.json

The repository includes a comprehensive `config.json` file with:

- **App Configuration**: Name, version, description
- **Server Settings**: Port, host, CORS, rate limiting
- **Database Configuration**: SQLite with connection pooling
- **UI Settings**: Theme colors, layout dimensions
- **Feature Flags**: Authentication, analytics, caching
- **Security Settings**: Headers, cookies, encryption
